How to compile:
Use command "javac HW1_weiming_liu.java" to compile

How to run:
Use command "java HW1_weiming_liu input_image_file S R A" to execute.
S = Scale ratio
R = Rotation angle in degree
A = Apply anti-aliasing or not; 1 = yes, 0 = no

Two windows will pop up. One shows the original image, and the other one shows the modified image.
